# Sistema de Cadastro de Produtos - Projeto MVC
Este projeto foi desenvolvido com o objetivo de aprender e praticar o padrão arquitetural **MVC (Model–View–Controller)** utilizando apenas HTML, CSS e JavaScript, sem banco de dados.
O sistema permite cadastrar produtos, exibí-los na tela e entender como cada camada do MVC se comporta dentro de um projeto simples.

## Objetivo do Projeto
- Aprender e praticar o padrão arquitetural MVC (Model–View–Controller).
- Entender a divisão de responsabilidades entre Model, View e Controller.

## Como o MVC funciona neste projeto

## *Model*
- Representa os dados do sistema.
- Ele define como um produto deve ser estruturado (nome, preço, descrição).

## *View*
- Responsável por exibir informações ao usuário.
- Neste projeto, a View contém funções que atualizam a interface.

## *Controller*
- Controla o fluxo do sistema.
- Ele recebe a ação do usuário (o envio do formulário), cria o produto, salva e manda atualizar a View.

## Funcionalidades
- Cadastro de produtos
- Exibição automática dos produtos cadastrados
- Código organizado seguindo o padrão MVC
- Interface simples e estilizada
- Armazenamento temporário em array (sem banco de dados)

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Padrão MVC aplicado no front-end
